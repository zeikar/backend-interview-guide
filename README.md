# 백엔드 개발자 면접 가이드

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![GitHub](https://img.shields.io/github/stars/zeikar/backend-interview-guide?style=social)](https://github.com/zeikar/backend-interview-guide)

<div class="guide-hero">
  <p class="guide-eyebrow">Backend Interview Guide</p>
  <p>대규모 트래픽을 처리하는 백엔드 시스템을 기준으로, 면접에서 자주 마주치는 주제를 카테고리별로 정리했습니다. 데이터베이스, 클라우드, 시스템 디자인, 프로그래밍을 한 흐름으로 훑을 수 있게 구성했습니다.</p>
</div>

한국어로 된 경력 개발자 면접 자료가 부족하여 개인 공부 겸 만들어 보았습니다. 문서는 Codex, Claude Code, Harness를 활용해 병렬로 작성·리뷰한 뒤 최종 검토를 거쳐 정리하였습니다.

오타나 수정, 추가할 내용이 있으면 언제든지 [PR](https://github.com/zeikar/backend-interview-guide/pulls) 부탁드립니다.

  <p>처음부터 전부 읽기보다 지원 포지션과 면접 스타일에 맞춰 카테고리별로 들어가는 편이 좋습니다. 설계형 면접이 많다면 시스템 디자인부터, 운영 감각을 보는 면접이 많다면 데이터베이스와 클라우드부터 보는 식으로 접근하면 됩니다.</p>
