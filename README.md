# 백엔드 개발자 면접 가이드

## 소개

백엔드 개발자 면접을 준비할 때 자주 나오는 주제와 설명을 정리한 문서입니다.

- GitHub 저장소: https://github.com/zeikar/backend-interview-guide

구글링 했을 때 한국어로 된 신입 개발자 면접 질문은 많으나 경력 개발자 면접 질문은 별로 없어서 개인 공부할 겸 만들어 보았습니다.

문서는 Codex, Claude Code, Harness를 활용해 병렬로 작성하고 리뷰한 뒤 정리하였으며, 모든 질문과 답변은 최종 검토 후 추가하였습니다.

오타나 수정, 추가할 내용이 있으면 언제든지 PR 부탁드립니다.
